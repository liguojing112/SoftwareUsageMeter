# SoftwareUsageMeter 客户使用说明

本文档面向**门店操作人员**，说明软件用途、操作流程和常见问题处理。

---

## 1. 软件用途

本软件配合"像素蛋糕"使用，用于对门店自助修图服务进行计时计费。

**工作原理：** 软件在后台监控像素蛋糕的运行状态，顾客每次进入导出流程时弹出收费框，管理员确认收款后顾客方可继续导出。

---

## 2. 计费方式

软件采用**混合计费**，同时考虑使用时长和导出张数：

```
总费用 = 使用时长（分钟）× 计时单价 + 导出张数 × 单张导出单价
```

**示例：**

- 使用 5 分钟，计时单价 2 元/分钟
- 导出 3 张，单张导出单价 4 元/张
- 合计：`5 × 2 + 3 × 4 = 22 元`

---

## 3. 主要功能

### 3.1 自动计时

- 顾客打开像素蛋糕，软件**自动开始计时**
- 顾客关闭像素蛋糕，软件**自动暂停计时**
- 不足 1 分钟按 1 分钟计算

### 3.2 自动识别导出

当前支持识别以下收费触发场景：

| 场景 | 说明 |
|------|------|
| 普通导出页 | 普通修图流程中的"快速导出 / 自定义设置" |
| 导出至本地 | 右上角"导出至本地"弹出的对话框 |
| 创意模块导出 | 创意模块中进入"导出至本地"后的对话框 |

> 说明："导出至创意"选择页本身不收费，真正收费的是后续的"导出至本地"对话框。

### 3.3 收费弹窗

收费框显示以下信息：

- 使用时长
- 计时单价
- 导出张数
- 单张导出单价
- 计时费用（小计）
- 导出费用（小计）
- **合计金额**（顾客应付总额）
- 微信/支付宝收款码

### 3.4 管理员确认

- 顾客**无法直接关闭**收费框
- 点击"确认收款"后，需输入**管理员密码**
- 密码正确后收费框关闭，顾客方可继续导出

## 4. 运行方式

```
dist\SoftwareUsageMeter.exe

```

## 5. 首次设置

### 5.1 打开管理后台

1. 程序启动后，系统托盘会显示图标
2. **右键**托盘图标
3. 选择"打开管理面板"
4. 输入管理员密码

> 默认管理员密码：`admin`（**首次使用后请务必修改**）

### 5.2 建议优先配置的内容

| 配置项 | 说明 | 建议值 |
|--------|------|--------|
| 计时单价 | 元/分钟 | 根据门店定价设置，如 `1.0` |
| 单张导出单价 | 元/张 | 根据门店定价设置，如 `1.0` |
| 默认导出张数 | OCR 失败时的兜底张数 | `1` |
| 收款码图片 | 微信/支付宝收款码 | 上传二维码图片 |
| 目标进程名 | 一般不需修改 | `PixCake.exe` |

### 5.3 修改管理员密码

进入管理后台后找到密码修改位置，输入新密码并保存。

> **重要：请勿继续使用默认密码 `admin`，建议设置为您自己易记但顾客不知道的密码。**

## 6. 日常使用流程

### 场景一：普通修图导出

1. 顾客打开像素蛋糕 → 软件自动开始计时
2. 顾客完成修图，进入导出页
3. 软件弹出收费框，显示金额
4. 顾客扫码付款
5. 管理员点击"确认收款"，输入密码
6. 收费框关闭
7. 顾客点击导出，完成本轮修图

### 场景二：导出至本地

1. 顾客点击"导出至本地"
2. 导出对话框出现
3. 软件弹出收费框
4. 管理员确认收款
5. 顾客继续完成导出

### 场景三：创意模块后导出至本地

1. 顾客进入创意模块
2. 后续进入"导出至本地"对话框
3. 软件弹出收费框
4. 管理员确认收款
5. 顾客完成导出

## 7. 导出张数识别说明

软件通过 Windows OCR 自动识别导出张数，识别区域包括：

1. 导出页左上角摘要区（如"导出 3 张图片"）
2. 本地导出对话框中的摘要区
3. 其他辅助区域作为兜底

**如果 OCR 识别失败**，软件会自动使用"默认导出张数"作为兜底值。建议将默认导出张数设置为门店最常见的导出数量（通常为 `1`）。

## 8. 常见问题

### Q1：为什么收费框没有弹出？

请依次检查：

1. 像素蛋糕是否正常运行？
2. 是否已经进入了导出流程（普通导出页 / 导出至本地）？
3. 管理后台中目标进程名是否为 `PixCake.exe`？
4. 日志文件（`app.log`）中是否有导出识别记录？

### Q2：为什么导出张数不准确？

可能原因：

- OCR 识别失败
- 导出对话框未完全显示
- 界面有遮挡

**临时处理：** 在管理后台将"默认导出张数"设为一个合理的兜底值。

### Q3：退出程序后像素蛋糕无法点击？

当前版本已加入自动恢复机制。如果仍出现异常，请：

1. **重新启动本程序一次**
2. 程序启动时会自动恢复像素蛋糕的交互状态

### Q4：如何修改管理员密码？

进入管理后台，找到密码修改位置，输入新密码并保存。
