# SoftwareUsageMeter

门店自助修图计时计费工具。监控"像素蛋糕"运行状态，导出时弹出收费框，管理员确认收款后恢复导出。

## 快速开始

**运行 exe：**
```
dist\SoftwareUsageMeter.exe
```

**运行源码：**
```powershell
pip install -r requirements.txt
python main.py
```

**打包：**
```powershell
pyinstaller build.spec --clean --noconfirm```

## 计费公式

```
总费用 = 使用时长（分钟）× 计时单价 + 导出张数 × 单张导出单价
```

## 核心功能

- 自动监控像素蛋糕进程启动与退出
- 自动计时，按分钟计费
- OCR 自动识别导出张数
- 全屏置顶收费弹窗
- 管理员密码确认收款
- 配置持久化保存

## 环境要求

- Windows 10 / 11
- Python 3.8+（源码运行时）
- 依赖：`PyQt5` / `psutil` / `pywin32` / `Pillow`

## 首次使用

1. 启动程序
2. 右键托盘图标 → 打开管理面板
3. 默认管理员密码：`admin`（**请及时修改**）
4. 配置计时单价、导出单价、收款码图片等
5. 保存后重启程序，进行完整测试
