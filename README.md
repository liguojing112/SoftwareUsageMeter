# 软件用途计时计费系统（SoftwareUsageMeter）

用于 Windows 场景下的桌面计时计费工具，面向门店自助修图设备等按时收费场景。  
程序可自动监控目标软件（默认 `PixCake.exe`）的运行时长，在检测到导出行为时弹出收费界面，确认收款后重置计时并继续下一轮。

## 功能特性

- 自动监控目标进程的启动与退出，自动开始/暂停计时
- 按分钟计费（不足 1 分钟按 1 分钟计）
- 检测导出窗口关键词，自动触发收费弹窗
- 全屏收费界面展示：使用时长、单价、应付金额、收款码
- 系统托盘常驻运行，支持状态查看和手动触发收费
- 管理面板支持费率、进程名、导出关键词、收款码、密码配置
- 支持单实例运行，避免重复启动

## 运行环境

- 操作系统：Windows 10 / 11
- Python：3.8 及以上（源码运行时）
- 依赖（见 `requirements.txt`）：
  - `PyQt5`
  - `psutil`
  - `pywin32`
  - `Pillow`
  - `pyinstaller`（用于打包）

## 快速开始

### 方式一：运行打包版本（推荐）

1. 在 `dist/` 目录找到 `SoftwareUsageMeter.exe`
2. 双击启动程序
3. 程序将在系统托盘运行

### 方式二：运行源码版本

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
2. 启动程序：
   ```bash
   python main.py
   ```

## 首次配置建议

1. 右键托盘图标，打开管理面板
2. 输入默认管理员密码：`admin`
3. 重点配置以下项目：
   - 每分钟费率（`rate`）
   - 目标进程名（默认 `PixCake.exe`）
   - 导出关键词（默认 `导出,Export`）
   - 收款码图片路径

## 计费与流程说明

1. 检测到目标进程运行后开始计时
2. 检测到导出行为后暂停计时并弹出收费界面
3. 操作员确认已支付后关闭弹窗
4. 计时重置并进入下一轮计费

计费公式：

```text
应付金额 = 使用分钟数 × 费率
```

## 配置文件说明

程序会在应用目录创建并读取 `config.json`，核心字段如下：

- `rate`: 每分钟单价（元）
- `admin_password`: 管理员密码哈希（SHA-256）
- `qr_code_path`: 收款码图片路径
- `process_name`: 目标进程名
- `export_window_keywords`: 导出窗口关键词列表
- `monitor_interval_ms`: 监控轮询间隔（毫秒）

## 常见问题

- 无法检测到目标软件：请确认进程名配置是否正确
- 未弹出收费界面：请检查导出关键词是否匹配目标窗口标题
- 忘记管理员密码：可手动编辑 `config.json` 中密码哈希（建议先备份）
