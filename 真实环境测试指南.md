# 软件用途计时计费系统 - 真实环境测试指南

## 概述
本指南提供在实际环境中测试软件用途计时计费系统的详细步骤。测试环境包括真实的像素蛋糕软件、Windows操作系统和门店使用场景。

## 测试环境准备

### 1. 硬件要求
- Windows 10/11 电脑
- 至少4GB内存
- 50MB可用磁盘空间
- 网络连接（用于下载像素蛋糕）

### 2. 软件要求
1. **像素蛋糕软件**：从官网下载最新版本
2. **测试用图片**：准备几张测试图片用于修图操作
3. **测试收款码**：准备微信/支付宝收款码图片

### 3. 测试文件准备
在项目目录下创建测试资源：
```bash
# 创建测试目录

# 准备测试图片（可从网上下载或使用自带图片）
# 准备收款码图片（可使用测试二维码生成器生成）
```

## 测试执行步骤

### 阶段1：基础功能验证

#### 1.1 程序启动测试
```bash

python main.py

# 方法2：使用打包后的EXE

```

**验证点：**
- [ ] 程序正常启动，无错误提示
- [ ] 系统托盘出现图标
- [ ] 右键菜单功能正常
- [ ] 状态窗口可正常显示/隐藏

#### 1.2 配置管理测试
1. 右键托盘图标 → "打开管理面板"
2. 输入默认密码：`admin`
3. 修改配置参数：

- [ ] 密码验证成功

## 常见问题排查

### 问题1：无法检测到像素蛋糕进程
**可能原因：**
1. 进程名不匹配
2. 像素蛋糕未以管理员权限运行
3. 防病毒软件拦截

**解决方案：**
1. 检查配置中的进程名是否为"PixCake.exe"
2. 以管理员身份运行像素蛋糕
3. 将程序添加到杀毒软件白名单

### 问题2：收费弹窗不显示
**可能原因：**
1. 导出关键词不匹配
2. 窗口识别失败
3. 显示设置问题

**解决方案：**
1. 在管理面板中添加更多关键词
2. 检查像素蛋糕窗口标题
3. 调整显示设置，确保支持全屏应用

### 问题3：计时不准确
**可能原因：**
1. 系统时间同步问题
2. 进程监控间隔设置不当
3. 程序性能问题

**解决方案：**
1. 同步系统时间
2. 调整监控间隔（默认2000ms）
3. 关闭不必要的后台程序

### 问题4：程序崩溃
**可能原因：**
1. 依赖库缺失
2. 系统兼容性问题
3. 资源不足

**解决方案：**
1. 重新安装依赖：`pip install -r requirements.txt`
2. 以兼容模式运行
3. 释放系统资源

## 自动化测试脚本

### 快速测试脚本
创建 `quick_test.py` 用于快速验证核心功能：

```python
#!/usr/bin/env python3
"""
快速测试脚本
用于快速验证核心功能
"""

import os
import sys
import time
import subprocess

def test_config():
    """测试配置管理"""
    print("测试配置管理...")
    # 实现配置测试逻辑
    return True

def test_timer():
    """测试计时器"""
    print("测试计时器...")
    # 实现计时器测试逻辑
    return True

def main():
    print("=" * 60)
    print("软件用途计时计费系统 - 快速测试")
    print("=" * 60)
    
    tests = [
        ("配置管理", test_config),
        ("计时器", test_timer),
        # 添加更多测试
    ]
    
    results = []
    for name, test_func in tests:
        print(f"\n执行测试: {name}")
        try:
            success = test_func()
            results.append((name, success))
            print(f"  {'✓' if success else '✗'} {name}")
        except Exception as e:
            print(f"  ✗ {name} - 错误: {e}")
            results.append((name, False))
    
    print("\n" + "=" * 60)
    print("测试结果:")
    print("=" * 60)
    
    passed = sum(1 for _, success in results if success)
    total = len(results)
    
    for name, success in results:
        status = "通过" if success else "失败"
        print(f"{name}: {status}")
    
    print(f"\n通过率: {passed}/{total} ({passed/total*100:.1f}%)")
    
    return passed == total

if __name__ == "__main__":
    success = main()
    sys.exit(0 if success else 1)
```
